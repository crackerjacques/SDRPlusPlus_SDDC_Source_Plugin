# Plugin Binary file installer


If you already have rnnoise( https://github.com/xiph/rnnoise.git )installed,

```
sudo dpkg -L sdrpp-rnnoise-plugin.deb
```

If not already installed

```
chmod +x install_rnnoise.sh
./install_rnnoise.sh
```

then

```
sudo dpkg -L sdrpp-rnnoise-plugin.deb
```
